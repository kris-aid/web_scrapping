
# 🕸️ Web Scraping para Investigadores: De Cero a Pro

Este repositorio contiene una guía práctica y desde cero para aprender **Web Scraping** utilizando Python. El objetivo es proporcionar a investigadores y estudiantes las herramientas necesarias para automatizar la recolección de datos de la web de forma ética y eficiente.

## 🎯 ¿Qué aprenderás con este Notebook?

Este tutorial está diseñado como un camino de aprendizaje que cubre:

1. **Fundamentos de la Web:** Entender HTML, CSS y el DOM.
2. **Herramientas Esenciales:**
* `Requests`: Para conectar con servidores y descargar páginas.
* `Regex`: Una mirada a la extracción manual de texto.
* `BeautifulSoup`: La librería estándar para navegar el HTML.


3. **Casos Prácticos:** * Extracción de vacantes de empleo en sitios estáticos.
* Scraping avanzado de una librería online con **paginación y extracción de calificaciones**.


4. **Gestión de Datos:** Cómo convertir información desordenada en archivos **CSV** listos para análisis con `Pandas`.
5. **Desafíos Reales:** Introducción a páginas dinámicas, logins, captchas y el uso de **APIs**.
6. **Ética y Legalidad:** Cómo ser un "buen ciudadano de la web" respetando los archivos `robots.txt` y términos de servicio.
